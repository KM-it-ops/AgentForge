Try it yourself

The YAML above is placeholder data. Clone the repo, edit spec/*.yaml with your name and paths, then emit platform-native config locally.

  1. Clone git clone https://github.com/KM-it-ops/AgentForge.git
  2. Edit spec Start with spec/identity.yaml — see CUSTOMIZE.md
  3. Emit npx @kmitops/agentforge init cursor --dir ./my-config
  4. Verify npm run verify from the repo root

Optional: run npm run demo locally for this page, or deploy Studio from the repo if you want your own compile playground — not required for the core workflow.
